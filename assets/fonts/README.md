# Bundled Yberium Pulse fonts

This directory contains the Latin and Latin Extended variable-font subsets used by the Yberium Pulse interface.

- Inter: `@fontsource-variable/inter@5.3.0`
- Manrope: `@fontsource-variable/manrope@5.3.0`
