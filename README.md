# VenueBrief Productisation Pack

Status: v1.7.1 has passed real-world pressure testing and is treated as Custom Stations Stable.

Reference versions:
- v1.5.1 - Simple Stable: simple use, no custom stations.
- v1.6 - Portable Stable: export/import JSON.
- v1.7.1 - Custom Stations Stable: advanced product candidate with custom stations.

Current rule:
Do not add new features. Freeze the working build, test it with 2-3 more real scenarios, then decide whether v1.7.1 becomes the main version.

Product line:
VenueBrief turns a rota into a clear operational shift brief in minutes.

Core positioning:
- Not a rota builder.
- Not payroll.
- Not HR software.
- Not a heavy SaaS.
- Lightweight shift-brief builder for hospitality operators.

Immediate next step:
Use this pack to present the product externally with neutral branding, fake data, a sample shift brief and a simple early-access offer.
